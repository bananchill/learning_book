# Символы (Symbol)

Символы — седьмой примитивный тип данных, добавленный в ES2015. Это уникальные идентификаторы, которые гарантированно не пересекаются друг с другом. Символы решают проблему конфликтов имён свойств и открывают доступ к метапрограммированию — настройке встроенного поведения объектов.

## О чём эта глава

Каждый вызов `Symbol()` создаёт абсолютно уникальное значение. Даже два символа с одинаковым описанием не равны друг другу. Это свойство делает символы идеальными ключами для свойств, которые не должны конфликтовать с другими свойствами объекта.

JavaScript также определяет набор встроенных (well-known) символов, которые позволяют изменять поведение объектов: делать их итерируемыми, настраивать приведение типов и проверку `instanceof`.

## Что вы узнаете

- Создание символов и их уникальность
- Использование символов как ключей объектов
- Глобальный реестр символов: `Symbol.for()` и `Symbol.keyFor()`
- Well-known символы: `Symbol.iterator`, `Symbol.toPrimitive`, `Symbol.hasInstance`
- Практические паттерны: приватные ключи, уникальные константы, метапрограммирование
