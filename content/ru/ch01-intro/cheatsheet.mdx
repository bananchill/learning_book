## JavaScript одной страницей

```js
// === ПОДКЛЮЧЕНИЕ ===
<script src="script.js" defer></script>  // лучший способ
<script src="analytics.js" async></script>  // независимые скрипты

// === СТРОГИЙ РЕЖИМ ===
'use strict'  // в начале файла (не нужен в ES-модулях)

// === ЗАПУСК ===
node script.js          // Node.js
node                    // REPL (интерактивный)
F12 → Console           // Консоль браузера

// === СОВРЕМЕННЫЙ JS (ES6+) ===
let x = 1              // переменная (можно переприсвоить)
const PI = 3.14        // константа (нельзя переприсвоить)

const double = n => n * 2              // стрелочная функция
const name = `Привет, ${user}!`       // шаблонный литерал
const { a, b } = obj                  // деструктуризация объекта
const [first, ...rest] = arr          // деструктуризация массива

const val = obj?.prop ?? 'default'    // optional chaining + nullish

async function load() {               // async/await
  const data = await fetch('/api')
  return data.json()
}
```

## Характеристики языка

| Свойство | JavaScript |
|---|---|
| Типизация | Динамическая |
| Наследование | Прототипное |
| Потоки | Однопоточный |
| Парадигмы | ООП, ФП, процедурный |
| Стандарт | ECMAScript (ECMA-262) |
| Версии | Ежегодно с ES2015 |

## Движки

| Среда | Движок |
|---|---|
| Chrome, Node.js, Edge | V8 (Google) |
| Firefox | SpiderMonkey (Mozilla) |
| Safari | JavaScriptCore (Apple) |

## Пайплайн выполнения (упрощённо)

```
Код → Parsing (AST) → Compilation → Execution → JIT-оптимизация
```

## Strict mode запрещает

- Использование необъявленных переменных
- Удаление переменных (`delete x`)
- Дублирующиеся параметры функций
- `this` в обычной функции = `undefined` (не `window`)
- Использование зарезервированных слов как имён
