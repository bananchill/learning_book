# Арифметические и строковые операторы

Прежде чем разбирать конкретные операторы, важно понять разницу между **выражением** и **инструкцией**. Выражение — это любой фрагмент кода, который вычисляется и возвращает значение: `2 + 3`, `x > 0`, `fn()`. Инструкция — это команда, которая выполняет действие: `if`, `for`, `let x = 5`. Операторы — это символы, которые превращают значения в выражения: берут один или несколько операндов и возвращают результат.

## Основные арифметические операторы

JavaScript предоставляет стандартный набор арифметических операторов. Все они работают с числами и возвращают число:

```javascript
10 + 3   // 13 — сложение
10 - 3   // 7  — вычитание
10 * 3   // 30 — умножение
10 / 3   // 3.3333... — деление (всегда float)
10 % 3   // 1  — остаток от деления (modulo)
10 ** 3  // 1000 — возведение в степень (ES2016)
```

## Оператор + и конкатенация строк

`+` — самый коварный оператор в JavaScript. Он выполняет **две совершенно разные операции**: сложение чисел и конкатенацию (склеивание) строк. Какая именно операция сработает, зависит от типов операндов. Если хотя бы один операнд — строка, JavaScript приведёт второй к строке и выполнит конкатенацию. Остальные арифметические операторы (`-`, `*`, `/`) всегда приводят к числу, поэтому подобных сюрпризов не создают.

```javascript
// Если один операнд — строка, происходит конкатенация
'Hello' + ' ' + 'World'  // 'Hello World'
'Цена: ' + 42            // 'Цена: 42' — число → строка
1 + 2 + '3'              // '33' — слева направо: 3, затем '33'
'1' + 2 + 3              // '123' — слева направо: '12', затем '123'
```

<Callout type="warning">
Порядок выполнения `+` имеет значение! `1 + 2 + '3' === '33'`, но `'3' + 1 + 2 === '312'`.
</Callout>

## Остаток от деления %

Оператор `%` возвращает остаток от целочисленного деления. Он незаменим в ситуациях, где нужна цикличность или проверка делимости:

```javascript
// Проверка чётности
5 % 2   // 1 — нечётное
4 % 2   // 0 — чётное

// Циклический индекс (оборот круга)
const colors = ['red', 'green', 'blue'];
colors[7 % 3]  // colors[1] = 'green'

// Часы: 25:00 → 1:00
25 % 24  // 1
```

## Приоритет операторов

Когда в одном выражении несколько операторов, JavaScript выполняет их не слева направо, а по **приоритету** — точно как в школьной математике, где умножение выполняется раньше сложения. Если приоритет одинаковый, срабатывает **ассоциативность** — направление вычисления (обычно слева направо, но `**` — справа налево):

```javascript
2 + 3 * 4      // 14, не 20 — * выполняется раньше +
(2 + 3) * 4    // 20 — скобки меняют порядок
2 ** 3 ** 2    // 512 — ** правоассоциативен: 2 ** (3 ** 2) = 2 ** 9
```

## Унарные операторы

Унарные операторы принимают один операнд. Самый полезный из них — унарный `+`, который приводит значение к числу. Это компактная альтернатива `Number()`, часто используемая при чтении данных из DOM или строковых значений:

```javascript
-5         // отрицание
+true      // 1 — принудительное приведение к числу
+'42'      // 42
+''        // 0
typeof x   // строка с типом
void 0     // undefined
```

## Инкремент и декремент

Операторы `++` и `--` увеличивают или уменьшают переменную на 1. Ключевое отличие — **когда** происходит изменение. Префиксная форма (`++n`) сначала изменяет значение и возвращает новое. Постфиксная (`n++`) возвращает старое значение и только потом изменяет переменную. В простых случаях разницы нет, но внутри выражений — результат будет разным:

```javascript
let n = 5;
n++   // постфиксный: возвращает 5, потом n = 6
++n   // префиксный: n = 7, возвращает 7
n--   // постфиксный: возвращает 7, потом n = 6
--n   // префиксный: n = 5, возвращает 5
```

<Callout type="tip">
На практике в 90% случаев используется постфиксная форма `i++` — например, в циклах `for`. Префиксная `++i` нужна, когда важно получить уже изменённое значение прямо в выражении.
</Callout>

<DeepDive title="Деление в JavaScript всегда float">

В отличие от многих языков (Java, C), в JavaScript нет целочисленного деления. `7 / 2 === 3.5`, а не `3`.

Если нужно целочисленное деление, используйте `Math.floor()`, `Math.trunc()` или `Math.ceil()`:

```javascript
Math.floor(7 / 2)   // 3 — округление вниз
Math.trunc(7 / 2)   // 3 — усечение дробной части
Math.ceil(7 / 2)    // 4 — округление вверх

// Для отрицательных чисел разница:
Math.floor(-7 / 2)  // -4
Math.trunc(-7 / 2)  // -3
```

</DeepDive>
