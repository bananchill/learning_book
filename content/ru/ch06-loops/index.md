# Циклы

Циклы позволяют выполнять повторяющиеся действия. JavaScript предоставляет несколько видов циклов для разных сценариев.

## Что вы узнаете

- `for`, `while`, `do...while` — когда каждый из них уместен
- `for...of` и `for...in` — итерация по значениям и ключам
- `break`, `continue` и метки — управление потоком
- Как писать эффективные циклы
