# Среда разработки

Хорошая среда разработки — это как качественный инструментарий для мастера. Правильно настроенный редактор, умение пользоваться браузерными инструментами и понимание экосистемы Node.js позволяют писать код быстрее, находить ошибки раньше и работать продуктивнее.

В этой главе мы настроим полноценное рабочее окружение JavaScript-разработчика с нуля.

<Callout type="info">
Даже если у вас уже настроена среда разработки, пробегитесь по главе — возможно, вы найдёте полезные расширения или настройки, которые сэкономят вам часы работы.
</Callout>

## Что вы узнаете

- Как настроить VS Code для комфортной работы с JavaScript и TypeScript
- Как использовать браузерный DevTools для отладки кода
- Как установить Node.js и работать с пакетными менеджерами
- Как настроить ESLint и Prettier для автоматического контроля качества кода
